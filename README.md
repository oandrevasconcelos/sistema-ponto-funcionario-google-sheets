# sistema-ponto-funcionario-google-sheets
Pipeline de Dados robusto que integra interfaces HTML a um ecossistema de produtividade (Google Workspace) via Make. A solução cobre desde a validação de entrada do usuário até o processamento lógico de timestamps e a entrega automatizada de relatórios gerenciais, eliminando falhas manuais e garantindo a confiabilidade dos dados para análise de BI.
